# Agenda Escolar

Aplicação pessoal de organização escolar feita com HTML, CSS e JavaScript puro.

## Funcionalidades

- Cadastro de atividades, trabalhos e avaliações
- Organização por prazos
- Tema claro e escuro
- Filtros de busca
- Checklist de subtarefas
- Edição rápida diretamente nos cards
- Backup por exportação e importação JSON
- Armazenamento local com localStorage

## Tecnologias

- HTML
- CSS
- JavaScript
- localStorage
